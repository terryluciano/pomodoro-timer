# Prompt
Create the markup for a todo list that will be to the left of the pomodoro timer in the App. The todo list should be a simple list of items that can be added, checked off, and deleted. The todo list should be styled to match the rest of the app, which is a simple, clean design with an ultra minimalist aesthetic. The todo list should be responsive and work well on both desktop and mobile devices. I have already created a todo store that you can reference.
